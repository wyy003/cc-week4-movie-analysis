# 电影票房数据分析工具

## 这个工具帮你做什么

把电影数据变成图表和结论，省得你手动画图、写分析报告、做PPT。

## 怎么用

1. 准备一个包含电影数据的Excel文件
2. 安装工具环境（需要技术支持）
3. 运行工具
4. 等待生成图表
5. 打开报告，查看结论

## 你能看到什么

### 5个核心结论

1. **票房冠军**：阿凡达以 2847.2 百万美元位居榜首
2. **地区分布**：全球发行电影占总票房的 83.8%，北美发行占 16.2%
3. **历史趋势**：2009 年因《阿凡达》创下最高平均票房记录
4. **类型表现**：科幻和动作类型电影在高票房榜单中表现突出
5. **数据覆盖**：数据集涵盖 1994-2022 年，总票房 44,515.5 百万美元

### 4张图表

1. **Top 10 票房电影排名** - 哪些电影最赚钱
2. **各地区票房占比** - 全球发行 vs 北美发行的对比
3. **年度平均票房趋势** - 1994-2022 年的票房变化
4. **各类型电影平均票房对比** - 科幻、动作、剧情等类型的表现

## 数据来源

- 数据文件：`movies_data.csv`
- 包含内容：电影名称、票房、地区、上映年份、类型、评分
- 样本数量：30 部高票房电影
